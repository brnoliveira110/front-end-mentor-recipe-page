# 🍳 Recipe Page

Este projeto é uma **landing page responsiva** de uma receita de omelete, desenvolvida como parte de um desafio da plataforma [Frontend Mentor](https://www.frontendmentor.io/).  

O objetivo principal foi praticar HTML semântico, responsividade, acessibilidade e estilização com Tailwind CSS, utilizando tecnologias modernas de desenvolvimento web como **Next.js** e **TypeScript**.

## ✅ Demonstração

🔗 Acesse o projeto online: [https://front-end-mentor-recipe-page-nine.vercel.app](https://front-end-mentor-recipe-page-nine.vercel.app)

## 🚀 Tecnologias Utilizadas

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## 📷 Layout

O layout foi baseado no design proposto pelo desafio do Frontend Mentor, respeitando as versões **mobile** e **desktop**.

## 🧩 Funcionalidades

Este projeto não possui funcionalidades dinâmicas — é uma **página estática** com foco em boas práticas de layout e estruturação.

## ⚙️ Como rodar localmente

1. Clone o repositório:
   ```bash
   git clone git@github.com:brnoliveira110/front-end-mentor-recipe-page.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Acesse no navegador:
   ```
   http://localhost:3000
   ```
